Go+_Goplus语言环境安装

2022-04-05

前两天了解到七牛云基于go开发的科学运算语言go+（goplus），便尝试想要搭建一个运行环境。找了一圈，只有这一篇知乎文章提到了，结果我还没成功：

E:\Programming\goplus>gop run e:\Programming\goplus\main.gop  
[FATAL] D:/gop/cmd/internal/run/run.go:91: findGoModFile: The system cannot find the file specified.

无奈只好作为小白第一次在github提了issue，没想到24小时就收到了大佬们的回复！作为感谢，也要把安装方法详细写出来。

本文参考资料：

  • Go+/GoPlus学习笔记教程--开发环境搭建图文(windows10) - 知乎
  • New to Gop, faced problem while installing... · Issue #821 · goplus/gop
  • gop/README.md at main · goplus/gop

一、安装go环境（需要1.16及以上）

go的安装很简单，官网下载即可，或者可以参考我原来的博文：

  • 2021.10最新测试Windows从go安装到beego第一个项目运行打包全过程（文末附beego官网示例无法运行的解决方案）_异想之旅的博客-CSDN博客（完成步骤二即可）
  • Linux Ubuntu安装Go语言环境_异想之旅的博客-CSDN博客

对于Linux安装，GitHub @xushiwei 在我的issue后面表示已经添加了all.bash，大家安装好go环境之后直接下载zip或者克隆仓库，然后执行all.bash即可（我的ubuntu系统go版本比较老还没测，但是给大佬端茶！！）

Windows请继续根据教程安装

二、使用go mod管理器安装gop

首先配置环境变量开启模块支持，然后echo查看自己的GOPATH：

set GO111MODULE=on
echo %GOPATH%

cd 进入 %GOPATH%/pkg/mod/github.com/goplus/gop@v1.0.13/cmd （其中v1.0.13换成你当前的版本）

执行如下语句：

go  install -v ./...
cd ..
gop install -v ./...

三、测试

这时候新建一个gop文件，输入官网样例运行试试：

echo println("Hello, world!") > main.gop
gop run main.gop

如果成功输出，那么恭喜你，否则请执行以下代码（未来再不成功也是如此）：

go get github.com/goplus/gop/builtin
gop run main.gop

完结撒花！

写完回顾下，上次没成功好像是把 gop install -v ./... 也写成 go install -v ./... 了……

祝七牛&gop越来越好啊！！
