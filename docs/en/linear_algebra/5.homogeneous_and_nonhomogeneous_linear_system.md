## Homogeneous Linear Systems

A linear system is called a ==homogeneous linear system (齐次线性方程组)== if it can be written in the form $A\vec{x} = \vec{0}$.

Obviously, a homogeneous linear system always has at least one solution $\vec{x} = \vec{0}$, which is called the ==trivial solution (平凡解)==. If there exists a solution $\vec{x} \neq \vec{0}$, we call it a ==nontrivial solution (非平凡解)==.

If a homogeneous linear system has a nontrivial solution, then it must have infinitely many solutions. This is also obvious because if $\vec{x}$ is a nontrivial solution, then for any scalar $c$, $c\vec{x}$ is also a solution.

A homogeneous linear system has a nontrivial solution if and only if there is at least one free variable. Otherwise, if every variable is basic, then there's only one solution, which is the trivial solution.

## Nonhomogeneous Linear Systems

A linear system is called a ==nonhomogeneous linear system (非齐次线性方程组)== if it can be written in the form $A\vec{x} = \vec{b}$ with $\vec{b} \neq \vec{0}$.

According to the definition of matrix–vector multiplication, we can conclude that $A\vec{x}$ has the same solution set as the vector equation $x_1\vec{v_1} + x_2\vec{v_2} + \cdots + x_n\vec{v_n} = \vec{b}$, where $\vec{v_1}, \vec{v_2}, \ldots, \vec{v_n}$ are the column vectors of $A$. The augmented matrix of the vector equation can be written as $\begin{bmatrix} \vec{v_1} & \vec{v_2} & \cdots & \vec{v_n} & \vec{b} \end{bmatrix}$.

The following statements are equivalent (either they are all true or all false):

- For each $\vec{b}$ in $\mathbb{R}^m$, the equation $A\vec{x} = \vec{b}$ has a solution.
- Each $\vec{b}$ in $\mathbb{R}^m$ is a linear combination of the columns of $A$.
- The columns of $A$ span $\mathbb{R}^m$.
- $A$ (NOT the augmented matrix) has a pivot position in every row.

Suppose $A\vec{x} = \vec{b}$ is consistent for some $\vec{b}$. Let $\vec{p}$ be a solution to $A\vec{x} = \vec{b}$. Then the solution set of $A\vec{x} = \vec{b}$ can be expressed as $\vec{x} = \vec{p} + \vec{v}$, where $\vec{v}$ is any solution of the corresponding homogeneous equation $A\vec{x} = \vec{0}$.
