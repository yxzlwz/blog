What is Linear Algebra?

To put it simple, Linear Algebra is the study of linear transformations.

Transformations

Transformation transforms an input to an output. In other words, a transformation is a function.

e.g., f(x)=2xf(x) = 2xf(x)=2x is a transformation that transforms input xxx to output 2x2x2x.

Linear Transformations (线性变换)

A transformation TTT that satisfies the linearity property:

  1. T(x+y)=T(x)+T(y)T(x + y) = T(x) + T(y)T(x+y)=T(x)+T(y)
  2. T(cx)=cT(x)T(cx) = cT(x)T(cx)=cT(x)

Matrix (矩阵)

Definition

Let

[m]×[n]={ (i,j)∣1≤i≤m,1≤j≤n,i,j∈Z }[m] \times [n] = \set{ (i, j) | 1 \leq i \leq m, 1 \leq j \leq n, i, j \in Z } [m]×[n]={(i,j)∣1≤i≤m,1≤j≤n,i,j∈Z}

then an m×nm \times nm×n (real) matrix is a function

A:[m]×[n]→RA: [m] \times [n] \to R A:[m]×[n]→R

We write the output by A(i,j)=ai,jA(i, j) = a_{i, j}A(i,j)=ai,j​, where

  • the input (i,j)(i, j)(i,j) represents the iiith row and jjjth column.
  • the output ai,ja_{i,j}ai,j​ is the number at the iiith row and jjjth column.
