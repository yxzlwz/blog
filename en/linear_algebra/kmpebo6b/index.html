Linear Equation (线性方程)

For n≥1n \geq 1n≥1, a linear equation in the variables x1,...,xnx_1, ... , x_nx1​,...,xn​ is an equation that can be written in the form

a1x1+a2x2+...+anxn=b a_1x_1 + a_2x_2 + ... + a_nx_n = b a1​x1​+a2​x2​+...+an​xn​=b

where bbb and the coefficients a1,...,ana_1, ... , a_na1​,...,an​ are real or complex numbers.

Examples

  • 2x1+πx2=0\sqrt{2}x_1 + \pi x_2 = 02​x1​+πx2​=0 is a linear equation.

  • x1x2+x3=0x_1x_2 + x_3 = 0x1​x2​+x3​=0 is NOT a linear equation because of the term x1x2x_1x_2x1​x2​.

  • x1+x2=0x_1 + \sqrt{x_2} = 0x1​+x2​​=0 is NOT a linear equation because of the term x2\sqrt{x_2}x2​​.

  • 0x1=00x_1 = 00x1​=0 is a linear equation.

  • 0x1=10x_1 = 10x1​=1 is a linear equation, although it has no solution.

Linear System (线性方程组)

Definition

A linear system is a collection of one or more linear equations involving the same variables (e.g., x1,...,xnx_1, ... , x_nx1​,...,xn​).

Solution of Linear Systems (线性方程组的解)

  • A solution of a linear system is a (ordered) list (s1,s2,...,sn)(s_1, s_2, ... ,s_n)(s1​,s2​,...,sn​) of numbers that makes each equation a true statement when the values s1,s2,...,sns_1, s_2, ... ,s_ns1​,s2​,...,sn​ are substituted for x1,...,xnx_1, ... , x_nx1​,...,xn​, respectively.

  • The set of all possible solutions is called the solution set of the linear system (线性方程组的解集).

  • A linear system may have:

    • no solution
    • exactly one solution
    • infinitely many solutions

  • A linear system that has at least one solution is called consistent (相容的); otherwise, it is called inconsistent (不相容的).

Equivalent Linear Systems (等价线性方程组)

Two linear systems are called equivalent if they have the same solution set.

Consistence (相容性)

A linear system is consistent if and only if the rightmost column of the augmented matrix is NOT a pivot column.

Simply put, a linear system is inconsistent if and only if the augmented matrix (in RREF) has a row of the form [00⋯0b]\begin{bmatrix}0 & 0 & \cdots & 0 & b\end{bmatrix}[0​0​⋯​0​b​], where bbb is a nonzero number.

Homogeneity

Homogeneous Linear Systems

A linear system is called a homogeneous linear system (齐次线性方程组) if it can be written in the form Ax⃗=0⃗A\vec{x} = \vec{0}Ax=0.

Obviously, a homogeneous linear system always has at least one solution x⃗=0⃗\vec{x} = \vec{0}x=0, which is called the trivial solution (平凡解). If there exists a solution x⃗≠0⃗\vec{x} \neq \vec{0}x=0, we call it a nontrivial solution (非平凡解).

If a homogeneous linear system has a nontrivial solution, then it must have infinitely many solutions. This is also obvious because if x⃗\vec{x}x is a nontrivial solution, then for any scalar ccc, cx⃗c\vec{x}cx is also a solution.

A homogeneous linear system has a nontrivial solution if and only if there is at least one free variable. Otherwise, if every variable is basic, then there's only one solution, which is the trivial solution.

Nonhomogeneous Linear Systems

A linear system is called a nonhomogeneous linear system (非齐次线性方程组) if it can be written in the form Ax⃗=b⃗A\vec{x} = \vec{b}Ax=b with b⃗≠0⃗\vec{b} \neq \vec{0}b=0.

According to the definition of matrix–vector multiplication, we can conclude that Ax⃗A\vec{x}Ax has the same solution set as the vector equation x1v1⃗+x2v2⃗+⋯+xnvn⃗=b⃗x_1\vec{v_1} + x_2\vec{v_2} + \cdots + x_n\vec{v_n} = \vec{b}x1​v1​​+x2​v2​​+⋯+xn​vn​​=b, where v1⃗,v2⃗,…,vn⃗\vec{v_1}, \vec{v_2}, \ldots, \vec{v_n}v1​​,v2​​,…,vn​​ are the column vectors of AAA. The augmented matrix of the vector equation can be written as [v1⃗v2⃗⋯vn⃗b⃗]\begin{bmatrix} \vec{v_1} & \vec{v_2} & \cdots & \vec{v_n} & \vec{b} \end{bmatrix}[v1​​​v2​​​⋯​vn​​​b​].

The following statements are equivalent (either they are all true or all false):

  • For each b⃗\vec{b}b in Rm\mathbb{R}^mRm, the equation Ax⃗=b⃗A\vec{x} = \vec{b}Ax=b has a solution.
  • Each b⃗\vec{b}b in Rm\mathbb{R}^mRm is a linear combination of the columns of AAA.
  • The columns of AAA span Rm\mathbb{R}^mRm.
  • AAA (NOT the augmented matrix) has a pivot position in every row.

Suppose Ax⃗=b⃗A\vec{x} = \vec{b}Ax=b is consistent for some b⃗\vec{b}b. Let p⃗\vec{p}p​ be a solution to Ax⃗=b⃗A\vec{x} = \vec{b}Ax=b. Then the solution set of Ax⃗=b⃗A\vec{x} = \vec{b}Ax=b can be expressed as x⃗=p⃗+v⃗\vec{x} = \vec{p} + \vec{v}x=p​+v, where v⃗\vec{v}v is any solution of the corresponding homogeneous equation Ax⃗=0⃗A\vec{x} = \vec{0}Ax=0.
