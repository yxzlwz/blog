Eigenvectors and Eigenvalues (特征向量和特征值)

Definition

An eigenvector of an n×nn \times nn×n matrix AAA is a nonzero vector x⃗\vec{x}x such that Ax⃗=λx⃗A\vec{x} = \lambda \vec{x}Ax=λx for some scalar λ\lambdaλ. The scalar λ\lambdaλ is called the eigenvalue corresponding to the eigenvector x⃗\vec{x}x.

Eigenvectors must be nonzero, while eigenvalues can be zero.

The set of all eigenvectors corresponding to the eigenvalue λ\lambdaλ is the eigenspace of AAA corresponding to λ\lambdaλ.

Finding Eigenvectors corresponding to an Eigenvalue

Let λ\lambdaλ be an eigenvalue of an n×nn \times nn×n matrix AAA. Then Ax⃗=λx⃗A\vec{x} = \lambda \vec{x}Ax=λx is equivalent to (A−λIn)x⃗=0⃗(A - \lambda I_n)\vec{x} = \vec{0}(A−λIn​)x=0. As a result, the eigenvectors corresponding to λ\lambdaλ are nontrivial solutions to the equation (A−λIn)x⃗=0⃗(A - \lambda I_n)\vec{x} = \vec{0}(A−λIn​)x=0.

A scalar λ\lambdaλ is an eigenvalue of an n×nn \times nn×n matrix AAA if and only if det(A−λIn)=0det(A - \lambda I_n) = 0det(A−λIn​)=0, which is called the characteristic equation (特征方程) of AAA.

Similarity (相似)

If AAA and BBB are n×nn \times nn×n matrices, then AAA is similar to BBB if there exists an invertible matrix PPP such that A=P−1BPA = P^{-1}BPA=P−1BP.

If AAA is similar to BBB, then AAA and BBB have the same characteristic polynomial, and thus the same eigenvalues with the same multiplicities. However, they may have different eigenvectors.

Diagonalization (对角化)

An n×nn \times nn×n matrix AAA is diagonal (对角矩阵) if all of its entries outside the main diagonal are zero.

A square matrix AAA is diagonalizable (可对角化) if AAA is similar to a diagonal matrix, i.e., if A=PDP−1A = PDP^{-1}A=PDP−1 for some diagonal matrix DDD and some invertible matrix PPP.
