Linear Transformations (线性变换)

Transformation transforms an input to an output. In other words, a transformation is a function.

E.g., f(x)=2xf(x) = 2xf(x)=2x is a transformation that transforms input xxx to output 2x2x2x.

A transformation TTT that satisfies the following linearity property is called a linear transformation:

  1. T(x+y)=T(x)+T(y)T(x + y) = T(x) + T(y)T(x+y)=T(x)+T(y)
  2. T(cx)=cT(x)T(cx) = cT(x)T(cx)=cT(x)

Matrix as a Linear Transformation

General Discussion

The equition Ax⃗=b⃗A\vec{x} = \vec{b}Ax=b can be viewed as a linear transformation that transforms input x⃗\vec{x}x to output b⃗\vec{b}b, and the matrix AAA represents this linear transformation, in other words, AAA is a function.

A transformation TTT from Rn\R^nRn to Rm\R^mRm is a function that takes an nnn-dimensional vector as input and produces an mmm-dimensional vector as output. RnR^nRn is called the domain (定义域) of TTT, and RmR^mRm is called the codomain (值域) of TTT.

For a given matrix AAA, we can define a linear transformation TTT from Rn\R^nRn to Rm\R^mRm as T(x⃗)=Ax⃗T(\vec{x}) = A\vec{x}T(x)=Ax.

One-to-one and Onto (单射和满射)

One-to-one (单射)

A linear transformation TTT from Rn\R^nRn to Rm\R^mRm is called one-to-one if for every b⃗\vec{b}b in the codomain Rm\R^mRm, there is at most one x⃗\vec{x}x in the domain Rn\R^nRn such that T(x⃗)=b⃗T(\vec{x}) = \vec{b}T(x)=b.

A linear transformation TTT from Rn\R^nRn to Rm\R^mRm is one-to-one if and only if the columns of the corresponding matrix AAA are linearly independent, which is equivalent to saying that AAA has a pivot in every column .

Onto (满射)

A linear transformation TTT from Rn\R^nRn to Rm\R^mRm is called onto if for every b⃗\vec{b}b in the codomain Rm\R^mRm, there is at least one x⃗\vec{x}x in the domain Rn\R^nRn such that T(x⃗)=b⃗T(\vec{x}) = \vec{b}T(x)=b.

A linear transformation TTT from Rn\R^nRn to Rm\R^mRm is onto if and only if the columns of the corresponding matrix AAA span Rm\R^mRm, which is equivalent to saying that AAA has a pivot in every row.

Discussion

If a linear transformation TTT from Rn\R^nRn to Rm\R^mRm is one-to-one, then n≤mn \leq mn≤m; if it is onto, then n≥mn \geq mn≥m. This is intuitive when thinking about the "number" of elements in the domain and codomain, while it can be proved by the above definitions and the properties of the corresponding matrix AAA.
